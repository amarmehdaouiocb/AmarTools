# AmarTools

Outils personnels pour le développement.

## Contenu

### [stt/](./stt/) - Speech to Text GUI

Interface graphique pour la transcription vocale avec Whisper large-v3 sur GPU.

- GUI minimaliste avec visualisation audio
- Accélération RTX/CUDA
- Focus intelligent (colle dans le bon terminal)

**Installation rapide :**
```powershell
cd stt
.\install.ps1
```

**Usage :**
```powershell
stt  # Lance le GUI
```
